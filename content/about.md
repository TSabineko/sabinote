このブログでは、おうちサーバーについてのあれこれや、Docker・Linux・ネットワークまわりの試行錯誤、調べたことを記録しています。

家の中で小さくサーバーを動かしたり、Docker でサービスを試したりしながら、個人で無理なく続けられる範囲の構築や運用を少しずつ整えています。

きれいにまとまった成功例だけでなく、途中で詰まったことや、実際に運用して分かったことも含めて残していくつもりです。あとから自分で見返して助かることと、同じように自宅環境を触る人の参考になることを大事にしています。

主に次のような内容を書いていく予定です。

- Ubuntu / Docker / 各種サービスの構築記録
- おうちサーバー運用で詰まった点と解決方法
- 実際に使ってよかった周辺機器や構成の見直し
- 調べたことのメモ・まとめ

こんな人に向けた内容を想定しています。

- おうちサーバーをこれから始めたい人
- Docker で自宅サービスを動かしてみたい人
- Linux やネットワーク設定で手を動かしながら覚えたい人
- 個人運用の記録やメモを読みたい人

実際に試した内容や失敗した点、運用して分かったことを重視して書いていきます。派手さよりも、あとから役に立つ記録を積み重ねていくブログにしていくつもりです。
